## 📋 留言须知

在留言之前，请阅读以下注意事项：

### ✅ 欢迎的内容

- 建设性的反馈和建议
- 真实的使用经验分享
- 准确的错误报告（请提供具体页面链接）
- 礼貌、尊重的交流

### ❌ 不欢迎的内容

- 广告、垃圾信息
- 违法违规内容
- 人身攻击、恶意诋毁
- 无关主题的讨论

### 🔒 隐私提醒

- 评论系统基于 GitHub Discussions，您的留言会公开显示
- 请**不要**在评论中分享您的 API Key、密码等敏感信息
- 留言前请确保您已登录 GitHub 账号

### ⏱️ 响应时间

我们会尽快查看和回复您的留言，但由于这是一个社区项目，可能无法立即回复所有问题。感谢您的理解和耐心！
