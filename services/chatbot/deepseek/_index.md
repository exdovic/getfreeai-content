## 🎯 服务简介

DeepSeek Chatbot 是一个功能强大的免费 AI 对话平台，提供联网搜索、代码执行、文件上传等高级功能，特别是 R1 模型的思维链展示功能独一无二。

**核心优势：**
- 🆓 **完全免费** - 每日 50 条消息，无需信用卡
- 🧠 **思维链可见** - R1 模型展示推理过程
- 🌐 **联网搜索** - 获取实时信息
- 💻 **代码执行** - 在线运行代码
- 📄 **文件上传** - 支持 PDF、Word、图片等
- 🇨🇳 **中文优化** - 中文性能顶尖

## 🚀 如何使用

### 前提条件

- ✅ 已注册 DeepSeek 账户（手机号或邮箱）
- ❌ 无需信用卡

详细注册步骤请参考：[DeepSeek 注册指南](/providers/deepseek#注册和账号)

### 使用步骤

#### 步骤 1：访问并登录

1. 访问：https://chat.deepseek.com
2. 使用账户登录

#### 步骤 2：选择模型

在对话界面选择模型：

| 模型 | 适用场景 | 特点 |
|------|---------|------|
| **DeepSeek V3** | 通用对话 | 中英文双语顶尖 |
| **DeepSeek R1** | 复杂推理 | 显示思维过程 |
| **DeepSeek Coder V2** | 代码任务 | 编程专家 |

#### 步骤 3：开始对话

1. 在输入框输入您的问题
2. 点击发送或按 Enter
3. 等待响应

## 🎨 功能特性

### 1. 基础对话

**支持功能：**
- 多轮对话，自动保持上下文
- 中英文双语
- 超长文本理解

### 2. 联网搜索（实时信息）

**功能：**
- 自动联网搜索最新信息
- 提供信息来源引用
- 确保答案时效性

**使用方法：**
- 直接询问需要实时信息的问题
- 例如："今天的新闻"、"最新的 AI 动态"

### 3. 代码执行

**功能：**
- 在线运行 Python、JavaScript 等代码
- 显示运行结果
- 支持数据可视化

**使用示例：**
```
用户："帮我画一个 sin 函数图像并运行"

DeepSeek 会：
1. 生成 Python 代码
2. 在沙箱中执行
3. 显示图像结果
```

### 4. 文件上传和分析

**支持格式：**
- 📄 **文档：** PDF、Word、Excel、TXT
- 🖼️ **图片：** JPG、PNG、GIF、WEBP
- 📊 **数据：** CSV、JSON

**使用方法：**
1. 点击输入框左侧的 📎 图标
2. 上传文件
3. 提出关于文件的问题

**应用场景：**
- 分析 PDF 论文
- 理解图片内容
- 处理 Excel 数据
- 总结 Word 文档

### 5. 思维链展示（R1 模型）

**功能：**
- 显示模型的推理过程
- 理解问题的解决步骤
- 学习思维方法

**示例：**
```
用户：计算 23 × 47

R1 思维过程：
1. 分解：23 × 47 = 20×47 + 3×47
2. 计算 20×47 = 940
3. 计算 3×47 = 141
4. 相加：940 + 141 = 1081

答案：1081
```

### 6. 图片理解（多模态）

**功能：**
- 识别图片内容
- 分析图表数据
- OCR 文字识别
- 图片描述生成

## 🔧 常见问题

### 1. 免费额度用完了怎么办？

**解决方法：**
- 等待第二天配额重置
- 或升级到 Pro 会员（付费）

### 2. 如何查看剩余配额？

**方法：**
- 在对话界面右上角查看
- 显示今日剩余消息数

### 3. R1 思维链显示慢？

**原因：**
- R1 模型需要更多思考时间
- 这是正常现象

**建议：**
- 耐心等待
- 复杂问题值得等待

### 4. 文件上传失败？

**检查：**
- 文件大小是否超过 10MB
- 文件格式是否支持
- 网络连接是否稳定
