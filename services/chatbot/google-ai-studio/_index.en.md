## ðŸš€ How to Use

### Prerequisites

- âœ… Registered Google account
- âœ… Accessed Google AI Studio and agreed to terms of service

For detailed registration steps, see: [Google AI Studio Registration Guide](/en/providers/google-ai-studio#registration-and-account)

### Usage Steps

#### Step 1: Access Chatbot Interface

1. Open browser and visit: https://aistudio.google.com
2. Login with Google account

#### Step 2: Select Model

Choose the model you want to use in the main interface:

| Model Name | Use Cases | Features |
|---------|---------|------|
| **Gemini 2.5 Flash** | Daily conversation, quick response | Latest version, fast speed |
| **Gemini 3 Flash** | High-frequency use | Fastest speed |
| **Gemini 2.5 Flash-Lite** | Simple tasks | Ultra-fast, lightweight |
| **Gemini Pro** | Complex tasks | Better understanding |

#### Step 3: Start Conversation

1. Enter your question in the input box
2. Click the attachment button to upload images, audio, or video
3. Click send or press Enter

#### Step 4: Adjust Parameters (Optional)

Click the settings icon on the right side to adjust:
- **Temperature:** Controls creativity (0-1)
- **Top-P:** Controls output diversity
- **Top-K:** Controls number of candidate words
- **Max Output Tokens:** Limits response length

## ðŸ”§ Common Issues

### 1. Can't Access Google AI Studio?

**Possible Causes:**
- Regional restrictions (Mainland China requires VPN)
- Browser compatibility issues
- Google account issues

**Solutions:**
- Use VPN tool
- Try Chrome or Edge browser
- Ensure Google account is working

### 2. Slow Response or Timeout?

**Possible Causes:**
- Unstable network connection
- Request content too long
- High server load

**Solutions:**
- Check network connection
- Reduce content per request
- Try again later

### 3. Image Upload Failed?

**Possible Causes:**
- Unsupported file format
- File too large (>4MB)
- Network issues

**Solutions:**
- Use supported formats (JPG, PNG, WEBP)
- Compress image size
- Check network connection

### 4. Response Quality Not Ideal?

**Optimization Methods:**
- Use more specific prompts
- Provide more contextual information
- Try different models
- Adjust temperature and other parameters
