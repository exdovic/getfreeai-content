## 🎯 服务简介

Google AI Studio 的 Chatbot 界面是一个强大的 Web 应用，让您无需编写任何代码即可使用 Gemini 系列模型。它提供了直观的对话界面和丰富的多模态功能。

**核心优势：**
- 🚀 **零门槛使用** - 无需编程知识
- 🎨 **多模态输入** - 支持文本、图像、音频、视频
- 🔍 **联网搜索** - 可以获取最新信息
- 💾 **Prompt 管理** - 保存和管理对话模板
- 🎛️ **参数调节** - 自定义温度、Top-P 等参数

## 🚀 如何使用

### 前提条件

- ✅ 已注册 Google 账户
- ✅ 已访问 Google AI Studio 并同意服务条款

详细注册步骤请参考：[Google AI Studio 注册指南](/providers/google-ai-studio#注册和账号)

### 使用步骤

#### 步骤 1：访问 Chatbot 界面

1. 打开浏览器，访问：https://aistudio.google.com
2. 使用 Google 账户登录

#### 步骤 2：选择模型

在主界面选择您想使用的模型：

| 模型名称 | 适用场景 | 特点 |
|---------|---------|------|
| **Gemini 2.5 Flash** | 日常对话、快速响应 | 最新版本，速度快 |
| **Gemini 3 Flash** | 高频使用 | 最快速度 |
| **Gemini 2.5 Flash-Lite** | 简单任务 | 超快、轻量级 |
| **Gemini Pro** | 复杂任务 | 更强理解能力 |

#### 步骤 3：开始对话

1. 在输入框中输入您的问题
2. 可以点击附件按钮上传图片、音频或视频
3. 点击发送或按 Enter 键

#### 步骤 4：调整参数（可选）

点击右侧的设置图标，可以调整：
- **Temperature（温度）：** 控制回答的创造性（0-1）
- **Top-P：** 控制输出的多样性
- **Top-K：** 控制候选词的数量
- **Max Output Tokens：** 限制回答长度

## 🎨 功能特性

### 1. 文本对话

**基础对话：**
- 支持中文、英文等多种语言
- 可以进行长对话，模型会记住上下文
- 支持最长 2M tokens 的超长上下文

**高级功能：**
- 代码生成和解释
- 文章写作和编辑
- 翻译和润色
- 数据分析和总结

### 2. 图像理解

**支持的功能：**
- 📷 **图像识别** - 识别图片中的物体、场景
- 📝 **图像描述** - 生成详细的图像描述
- 🔍 **OCR 文字识别** - 提取图片中的文字
- 📊 **图表分析** - 理解和解释图表数据
- 🎨 **图像比较** - 比较多张图片的异同

**使用方法：**
1. 点击输入框旁的 📎 图标
2. 选择要上传的图片（支持 JPG, PNG, WEBP 等格式）
3. 输入您的问题（例如："这张图片里有什么？"）
4. 发送请求

### 3. 音频处理

**支持的功能：**
- 🎵 **音频转文字** - 将语音转换为文本
- 🎼 **音频内容理解** - 理解音频中的信息
- 🗣️ **多语言识别** - 支持多种语言的音频

**支持格式：** MP3, WAV, AAC 等

### 4. 视频分析

**支持的功能：**
- 🎬 **视频内容总结** - 总结视频的主要内容
- 🔍 **场景识别** - 识别视频中的场景和动作
- 📝 **视频转录** - 提取视频中的语音文字

**支持格式：** MP4, MOV, AVI 等

### 5. Google Search Grounding

**功能说明：**
- 模型可以实时搜索 Google 获取最新信息
- 回答会包含信息来源链接
- 确保信息的时效性和准确性

**使用方法：**
1. 在设置中启用 "Grounding with Google Search"
2. 提问时模型会自动搜索相关信息
3. 回答中会显示引用的来源

**适用场景：**
- 查询最新新闻和事件
- 获取当前天气、股票等实时信息
- 查找最新的技术文档和教程

### 6. Prompt 管理

**功能：**
- 💾 **保存 Prompt** - 将常用的提示词保存为模板
- 📝 **编辑模板** - 修改和优化已保存的 Prompt
- 🔄 **复用模板** - 快速应用已保存的 Prompt
- 📂 **分类管理** - 组织和管理多个 Prompt

**使用场景：**
- 创建标准化的提问模板
- 团队协作时共享 Prompt
- A/B 测试不同的 Prompt 效果

## 🔢 使用限制

### Chatbot 使用配额

虽然 Chatbot 共享 API 配额，但在实际使用中限制较为宽松：

| 限制类型 | 配额 | 说明 |
|---------|------|------|
| 每日 Tokens | 15M tokens | 与 API 共享 |
| 每日请求数 | 1,500 次 | 与 API 共享 |
| 单次上下文 | 2M tokens | 超长上下文支持 |
| 文件大小 | 视类型而定 | 图片: 4MB, 音频/视频: 100MB |

### 实际使用体验

对于 Chatbot 使用者，配额通常足够：
- ✅ 每天可以进行数百次对话
- ✅ 支持长时间连续对话
- ✅ 多模态文件上传不受特殊限制

## 💡 使用技巧

### ✅ 最佳实践

1. **选择合适的模型**
   - 日常对话：Gemini 2.5 Flash
   - 复杂任务：Gemini Pro
   - 快速响应：Gemini 2.5 Flash-Lite

2. **充分利用多模态**
   - 上传图片帮助模型理解
   - 使用语音输入提高效率
   - 分析视频内容节省时间

3. **优化 Prompt**
   - 明确具体的问题
   - 提供必要的上下文
   - 使用示例说明期望的输出

4. **利用 Grounding**
   - 需要最新信息时启用搜索
   - 验证回答的准确性
   - 查看引用来源

5. **管理对话历史**
   - 保存重要的对话
   - 创建 Prompt 模板
   - 定期清理不需要的历史

### ⚠️ 注意事项

1. **隐私保护**
   - 不要上传敏感个人信息
   - 可以选择退出数据训练
   - 定期检查隐私设置

2. **配额管理**
   - 注意每日配额限制
   - 避免无意义的重复请求
   - 使用合适的模型节省配额

3. **网络要求**
   - 部分地区需要科学上网
   - 确保网络连接稳定
   - 上传大文件时注意网速

4. **内容政策**
   - 遵守 Google 的使用政策
   - 不要请求生成有害内容
   - 尊重版权和知识产权

## 🔧 常见问题

### 1. 无法访问 Google AI Studio？

**可能原因：**
- 地区限制（中国大陆需科学上网）
- 浏览器兼容性问题
- Google 账户问题

**解决方法：**
- 使用科学上网工具
- 尝试 Chrome 或 Edge 浏览器
- 确保 Google 账户正常

### 2. 回答速度慢或超时？

**可能原因：**
- 网络连接不稳定
- 请求内容过长
- 服务器负载高

**解决方法：**
- 检查网络连接
- 减少单次请求的内容量
- 稍后重试

### 3. 图片上传失败？

**可能原因：**
- 文件格式不支持
- 文件过大（>4MB）
- 网络问题

**解决方法：**
- 使用支持的格式（JPG, PNG, WEBP）
- 压缩图片大小
- 检查网络连接

### 4. 回答质量不理想？

**优化方法：**
- 使用更明确的 Prompt
- 提供更多上下文信息
- 尝试不同的模型
- 调整温度等参数
